🧪 Test Mode vs Real Mode

Test mode allows you to safely test changes without actually writing to hardware.

⚠️ Always Test First!

  • Enable TestMode before any new changes
  • Review logs to ensure validation passes
  • Only then disable TestMode and apply for real

💡 Format Explanation

Format: 0x[P-cores][E-cores]

Example: 0x0406 = 4 P-cores + 6 E-cores

  • Upper byte (04) = P-cores
  • Lower byte (06) = E-cores
